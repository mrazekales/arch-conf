# myArch Installer
#### Install and config base Arch system
```bash
$ pacman -Sy git
$ git clone git://github.com/mrazekales/myarch-installer.git
$ cd myarch-installer
$ sh arch_install 
```
#### Install and config i3wm+LightDM
```bash
$ sh i3wm_install
```
