# myArch Installer
**Run script on booted Arch liveiso**
```bash
$ pacman -Sy git
$ git clone git://github.com/mrazekales/myarch-installer.git
$ cd myarch-installer
$ sh arch_install
$ sh 
```

## Arch Base Installation

```bash
$ ./arch_install
```

## Desktop Environment Installation 

```bash
$ ./de_install
```

## Apps installation

```bash
$ ./apps_install
```
